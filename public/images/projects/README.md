# Project Logos

Place your project logo PNG files here.

## Naming Convention

Name your logo files to match the `logo` field in `src/data/projects.js`:
- `project-one.png`
- `project-two.png`
- `project-three.png`
- etc.

## Logo Specifications

- **Format**: PNG (with transparency recommended)
- **Recommended Size**: 200x160px or similar aspect ratio
- **Max Display Size**: 100px width Ã— 80px height (will be scaled automatically)
- **Background**: Transparent PNGs work best

The logos will be displayed at the top of each project card.
